# ITAM Sistema de Actas

+ Instalar las dependencias
    $ cd 
    $ npm install

+ Instalar sass (windows)

    $ gem install sass
    
+ Correr el webserver

    $ gulp serve
    
+ Login

	No hay ususarios, continuar sin restricciones.

## Construccion 
  
    $ gulp build
    $ gulp deploy
